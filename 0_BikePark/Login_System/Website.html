BikePark

Was ist Bike Park?

BikePark bietet Ihnen eine Möglichkeit Ihr Fahrrad sicher und einfach abzustellen.

Wie können Sie BikePark verwenden?

Sie können sich auf unserer Website einen Account erstellen. Wenn Sie bei einem unserer BikeParks sind, melden Sie sich mit Ihrem Account an und scannen Ihren QR-Code. Danach können Sie sich einen beliebigen freien Platz reservieren.
